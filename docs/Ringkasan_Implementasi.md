# Ringkasan Implementasi

MVP ini dibangun sebagai aplikasi web statis pendamping Google Sheet dan Google Drive.

Yang sudah tersedia:

- Seed data dari workbook Juni 2026.
- Dashboard KPI dan grafik.
- Tabel paket dengan pencarian dan filter.
- Form tambah paket dengan validasi dan penyimpanan Draft Lokal.
- Tabel anggaran, POK detail, penyedia, dokumen, checklist, register nomor, dan arsip.
- Halaman laporan paket yang bisa dicetak.
- Pengaturan endpoint Apps Script dan Root Folder ID di localStorage.
- Folder `apps-script` berisi sumber Google Apps Script dari arsip lampiran.

Yang sengaja belum dibuat pada MVP:

- Backend server tambahan.
- Database eksternal.
- Upload file langsung dari frontend.
- Penyimpanan token atau secret di frontend.
